# 侧边栏小工具管理系统 - 功能说明

## 已实现的功能

### 1. 核心功能
- ✅ 小工具配置管理系统（启用/禁用）
- ✅ 小工具拖拽排序（上移/下移按钮）
- ✅ 页面级别配置（首页、文章列表、文章详情）
- ✅ 小工具自定义设置面板
- ✅ 配置持久化到数据库
- ✅ 动态侧边栏渲染

### 2. 支持的小工具
- 博主卡片 (BloggerCard)
- 热门文章 (PopularArticles)
- 标签云 (TagCloud)
- 广告卡片 (AdCard)
- 倒计时 (CountdownCard)
- 天气卡片 (WeatherCard)

### 3. 数据库集成
- 使用现有的 Setting 模型存储配置
- Key: `sidebar_widgets_config`
- Type: JSON
- Group: website

### 4. 管理界面
- 位置：后台 > 设置 > 小工具管理
- 功能：
  - 按页面切换配置
  - 启用/禁用小工具
  - 调整显示顺序
  - 自定义小工具设置
  - 恢复默认配置
  - 保存配置

## 如何使用

### 访问小工具管理
1. 登录后台：http://localhost:3000/zh/admin
2. 导航到：设置 > 小工具管理
3. 选择要配置的页面类型（首页/文章列表/文章详情）
4. 配置小工具：
   - 点击眼睛图标启用/禁用
   - 使用上下箭头调整顺序
   - 点击齿轮图标展开设置面板
5. 点击"保存配置"保存更改

### 配置结构

```json
{
  "home": [
    {
      "id": "blogger",
      "enabled": true,
      "order": 0,
      "settings": {}
    }
  ],
  "articles": [...],
  "article_detail": [...]
}
```

## 文件结构

```
apps/web/src/
├── lib/
│   └── sidebar-config.ts          # 小工具配置定义和辅助函数
├── components/
│   ├── DynamicSidebar.tsx         # 动态侧边栏组件
│   └── admin/
│       └── WidgetManager.tsx      # 小工具管理 UI 组件
├── app/
│   ├── actions/
│   │   └── widget-config.ts       # Server Actions (保存/读取配置)
│   └── [locale]/
│       ├── page.tsx               # 首页 - 使用 DynamicSidebar
│       └── articles/
│           ├── page.tsx           # 文章列表 - 使用 DynamicSidebar
│           └── [slug]/page.tsx    # 文章详情 - 使用 DynamicSidebar
```
