# 🚀 快捷命令安装指南

## 📋 功能说明

像宝塔面板的 `bt` 命令一样，现在您可以通过 `blog` 命令快速唤起部署菜单！

---

## 🎯 快速安装

### 方法 1: 自动安装（推荐）

```bash
# 运行安装脚本
bash install-blog-cmd.sh
```

安装脚本会自动：
- ✅ 检测您的 Shell 类型（bash/zsh）
- ✅ 创建符号链接到系统路径
- ✅ 配置环境变量
- ✅ 提供多种安装选项

### 方法 2: 手动安装

```bash
# 1. 创建符号链接（需要 sudo 权限）
sudo ln -s /Volumes/文档/blog/blog /usr/local/bin/blog

# 2. 验证安装
blog 3
```

### 方法 3: 添加到 PATH

编辑 `~/.zshrc` 或 `~/.bashrc`，添加：

```bash
export PATH="/Volumes/文档/blog:$PATH"
```

然后刷新配置：

```bash
source ~/.zshrc  # 或 source ~/.bashrc
```

## 🚨 故障排除

### 问题 1: `blog: command not found`

**解决方法：**

```bash
# 检查是否安装成功
ls -l /usr/local/bin/blog

# 如果没有，手动安装
sudo ln -s /Volumes/文档/blog/blog /usr/local/bin/blog

# 或者添加到 PATH
echo 'export PATH="/Volumes/文档/blog:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### 问题 2: 权限错误

**解决方法：**

```bash
# 确保脚本有执行权限
chmod +x /Volumes/文档/blog/blog
chmod +x /Volumes/文档/blog/install-blog-cmd.sh
chmod +x /Volumes/文档/blog/deploy.sh
```

### 问题 3: 路径包含空格

如果您的路径包含空格（如 `/Volumes/文档/blog`），某些系统可能无法正确识别。

**解决方法：**

```bash
# 方法 1: 使用引号
export PATH="/Volumes/文档/blog:$PATH"

# 方法 2: 使用转义字符
export PATH=/Volumes/文档/blog:$PATH

# 方法 3: 创建符号链接到不含空格的路径
sudo ln -s "/Volumes/文档/blog/blog" /usr/local/bin/blog
```
